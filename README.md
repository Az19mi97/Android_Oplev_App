# The Android Application
OPLEV is a travel-planning app concept designed to make it easy to organize, save, and share trip ideas with others.  
The app brings wishes, notes, images, and collaboration together in one place.

## Features to Consider

### 1. Account Creation & Login
- Users can create an account.
- Secure login system with database support.
- Stores user data safely.

### 2. Travel Wishlist / Folder System
- Create folders for destinations or travel ideas.
- Add items, plans, to-dos, and links.
- Organize content by destination or theme.

### 3. Notes for the Travel Plan
- Write notes directly inside each folder.
- Support for text, checklists, and descriptions.

### 4. Image Upload
- Upload images to each folder.
- Image gallery view.
- Stored via cloud or database.

### 5. Sharing & Collaboration
- Share folders with other users in the app.
- All invited users can:
  - Add notes  
  - Edit content  
  - Upload images  
  - Update the plan  
- Ideal for group trips and collaborative planning.
